## ✨ Features

* 🛍️ Browse a wide range of products with an attractive UI
* 🔍 Search products instantly by name
* 📂 Filter products by categories
* ❤️ Add and manage favorite products
* 🛒 Add products to cart with quantity management
* ➕ Increase or decrease product quantity
* 💰 Automatic total price calculation
* 📦 Order summary and checkout functionality
* 📱 Fully responsive design for mobile, tablet, and desktop
* ⭐ Product ratings and reviews display
* 🚚 Delivery charge calculation
* 📜 Order history management
* 🔄 Dynamic product data using API integration
* 🎨 Modern and user-friendly interface
* ⚡ Fast and smooth user experience
* 🔐 Local Storage support for cart, favorites, and orders
